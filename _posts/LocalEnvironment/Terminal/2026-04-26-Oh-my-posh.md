## Used tools and versions

- **PowerShell Version:** 7.6.1

## Font

1. Download font from [https://www.nerdfonts.com/](https://www.nerdfonts.com/){:target="_blank"} for example **CascadiaCode**
2. Install font from guide [https://ohmyposh.dev/docs/installation/fonts](https://ohmyposh.dev/docs/installation/fonts){:target="_blank"}

## Windows

```powershell
winget install JanDeDobbeleer.OhMyPosh -s winget
```

Restart terminal

```powershell
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\jandedobbeleer.omp.json" | Invoke-Expression
Install-Module -Name Terminal-Icons -Force
Install-Module -Name PowerShellGet -Force
Install-Module PSReadLine -AllowPrerelease -Force
```

Modify configuration file

```powershell
notepad $PROFILE

oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\atomic.omp.json" | Invoke-Expression
Import-Module Terminal-Icons
Set-PSReadLineOption -PredictionSource HistoryAndPlugin
Set-PSReadLineOption -PredictionViewStyle ListView
```

## Linux

```bash
sudo apt-get install --reinstall unzip
sudo curl -s https://ohmyposh.dev/install.sh | bash -s
cd ~/
```

Modify configuration file

```bash
nano .bashrc

export PATH=$PATH:/home/pokrz26/.local/bin
eval "$(oh-my-posh init bash --config /mnt/c/Users/mpokrzywinski/AppData/Local/Programs/oh-my-posh/themes/atomic.omp.json)"

KEY: ctrl+o (Save)
KEY: ctrl+x (Close)
COMMAND: exec bash
```
